–≠–≤–æ–ª—é—Ü–∏—è API –±–µ–∑ –∫–æ–Ω—Ç—Ä–æ–ª—è –≤–µ—Ä—Å–∏–π#

–†–∞–∑–≤–∏—Ç–∏–µ API –≤–ª–µ—á–µ—Ç –∑–∞ —Å–æ–±–æ–π –ø—Ä–æ–±–ª–µ–º—É —Å–æ—Ö—Ä–∞–Ω–µ–Ω–∏—è —Å—Ç–∞—Ä–æ–π –≤–µ—Ä—Å–∏–∏ –¥–æ —Ç–µ—Ö –ø–æ—Ä, –ø–æ–∫–∞ —Ä–∞–∑—Ä–∞–±–æ—Ç—á–∏–∫–∏ –Ω–µ –ø–µ—Ä–µ–π–¥—É—Ç –Ω–∞ –Ω–æ–≤—É—é. –ò—Ç–∞–∫, —Å REST –ø—Ä–∏–Ω—è—Ç–æ –ø—Ä–µ–¥–ª–∞–≥–∞—Ç—å –Ω–µ—Å–∫–æ–ª—å–∫–æ –≤–µ—Ä—Å–∏–π API. –¢–µ–º –Ω–µ –º–µ–Ω–µ–µ, GraphQL —É—Å—Ç—Ä–∞–Ω—è–µ—Ç –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ—Å—Ç—å –≤ —É–ø—Ä–∞–≤–ª–µ–Ω–∏–∏ –≤–µ—Ä—Å–∏—è–º–∏, –æ—Ç–∫–∞–∑—ã–≤–∞—è—Å—å –æ—Ç API –Ω–∞ —É—Ä–æ–≤–Ω–µ –ø–æ–ª–µ–π. –£—Å—Ç–∞—Ä–µ–≤—à–∏–µ –ø–æ–ª—è –º–æ–≥—É—Ç –±—ã—Ç—å –ø–æ–∑–∂–µ —É–¥–∞–ª–µ–Ω—ã –∏–∑ —Å—Ö–µ–º—ã, –Ω–µ –≤–ª–∏—è—è –Ω–∞ —Å—É—â–µ—Å—Ç–≤—É—é—â–∏–µ –∑–∞–ø—Ä–æ—Å—ã. GraphQL –¥–µ–ª–∞–µ—Ç —ç—Ç–æ –≤–æ–∑–º–æ–∂–Ω—ã–º –ø—É—Ç–µ–º —Å–æ–∑–¥–∞–Ω–∏—è –µ–¥–∏–Ω–æ–≥–æ API –¥–ª—è –≤—Å–µ–≥–æ –ø—Ä–∏–ª–æ–∂–µ–Ω–∏—è, –∫–æ—Ç–æ—Ä–æ–µ –Ω–µ –æ–≥—Ä–∞–Ω–∏—á–µ–Ω–æ –∫–æ–Ω–∫—Ä–µ—Ç–Ω—ã–º –º–µ—Ö–∞–Ω–∏–∑–º–æ–º —Ö—Ä–∞–Ω–µ–Ω–∏—è.

–ò—Å–ø–æ–ª—å–∑—É—è –µ–¥–∏–Ω—É—é —Ä–∞–∑–≤–∏–≤–∞—é—â—É—é—Å—è –≤–µ—Ä—Å–∏—é#

API-–∏–Ω—Ç–µ—Ä—Ñ–µ–π—Å—ã GraphQL –ø—Ä–µ–¥–æ—Å—Ç–∞–≤–ª—è—é—Ç –ø—Ä–∏–ª–æ–∂–µ–Ω–∏—è–º –ø–æ—Å—Ç–æ—è–Ω–Ω—ã–π –¥–æ—Å—Ç—É–ø –∫ –Ω–æ–≤—ã–º —Ñ—É–Ω–∫—Ü–∏—è–º –∏ —Å–ø–æ—Å–æ–±—Å—Ç–≤—É—é—Ç —Å–æ–∑–¥–∞–Ω–∏—é –±–æ–ª–µ–µ –ø–æ–Ω—è—Ç–Ω–æ–≥–æ –∏ –±–æ–ª–µ–µ —É–¥–æ–±–Ω–æ–≥–æ –≤ –æ–±—Å–ª—É–∂–∏–≤–∞–Ω–∏–∏ —Å–µ—Ä–≤–µ—Ä–Ω–æ–≥–æ –∫–æ–¥–∞.

–°–æ–≤–º–µ—Å—Ç–Ω–æ–µ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ –∫–æ–¥–∞#

–í GraphQL –ø–æ–ª—è, –∏—Å–ø–æ–ª—å–∑—É–µ–º—ã–µ –≤ –Ω–µ—Å–∫–æ–ª—å–∫–∏—Ö –∑–∞–ø—Ä–æ—Å–∞—Ö, –º–æ–≥—É—Ç —Å–æ–≤–º–µ—Å—Ç–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å—Å—è –Ω–∞ –±–æ–ª–µ–µ –≤—ã—Å–æ–∫–æ–º —É—Ä–æ–≤–Ω–µ –∫–æ–º–ø–æ–Ω–µ–Ω—Ç–æ–≤ –¥–ª—è –ø–æ–≤—Ç–æ—Ä–Ω–æ–≥–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è. –≠—Ç–∞ —Ñ—É–Ω–∫—Ü–∏—è, –Ω–∞–∑—ã–≤–∞–µ–º–∞—è —Ñ—Ä–∞–≥–º–µ–Ω—Ç–∞–º–∏, –ø–æ–∑–≤–æ–ª—è–µ—Ç –ø–æ–ª—É—á–∞—Ç—å —Ä–∞–∑–Ω—ã–µ –¥–∞–Ω–Ω—ã–µ, —Å–æ—Ö—Ä–∞–Ω—è—è –æ–¥–Ω–æ –∏ —Ç–æ –∂–µ –ø–æ–ª–µ —Å—Ö–µ–º—ã.

–ü–æ–¥—Ä–æ–±–Ω—ã–µ —Å–æ–æ–±—â–µ–Ω–∏—è –æ–± –æ—à–∏–±–∫–∞—Ö#

–í REST –º—ã –ø—Ä–æ—Å—Ç–æ –ø—Ä–æ–≤–µ—Ä—è–µ–º –∑–∞–≥–æ–ª–æ–≤–∫–∏ HTTP –Ω–∞ —Å—Ç–∞—Ç—É—Å –æ—Ç–≤–µ—Ç–∞, –æ—Å–Ω–æ–≤—ã–≤–∞—è—Å—å –Ω–∞ –∫–æ—Ç–æ—Ä–æ–º –º—ã –º–æ–∂–µ–º –æ–ø—Ä–µ–¥–µ–ª–∏—Ç—å, —á—Ç–æ –ø–æ—à–ª–æ –Ω–µ —Ç–∞–∫ –∏ –∫–∞–∫ —Å –Ω–∏–º —Å–ø—Ä–∞–≤–∏—Ç—å—Å—è. –ò –Ω–∞–æ–±–æ—Ä–æ—Ç, –µ—Å–ª–∏ –ø—Ä–∏ –æ–±—Ä–∞–±–æ—Ç–∫–µ –∑–∞–ø—Ä–æ—Å–æ–≤ GraphQL –≤–æ–∑–Ω–∏–∫–∞–µ—Ç –æ—à–∏–±–∫–∞, —Å–µ—Ä–≤–µ—Ä–Ω–∞—è —á–∞—Å—Ç—å –ø—Ä–µ–¥–æ—Å—Ç–∞–≤–∏—Ç –ø–æ–¥—Ä–æ–±–Ω–æ–µ —Å–æ–æ–±—â–µ–Ω–∏–µ –æ–± –æ—à–∏–±–∫–µ, –≤–∫–ª—é—á–∞—é—â–µ–µ –≤—Å–µ —Ä–∞—Å–ø–æ–∑–Ω–∞–≤–∞—Ç–µ–ª–∏ –∏ —Å—Å—ã–ª–∞—é—â—É—é—Å—è –Ω–∞ –∫–æ–Ω–∫—Ä–µ—Ç–Ω—É—é —á–∞—Å—Ç—å –∑–∞–ø—Ä–æ—Å–∞ –ø—Ä–∏ –æ—à–∏–±–∫–µ. –°–æ–æ–±—â–µ–Ω–∏—è –æ–± –æ—à–∏–±–∫–∞—Ö GraphQL –Ω–µ –∏–º–µ—é—Ç –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω–æ–≥–æ —Å—Ç–∞–Ω–¥–∞—Ä—Ç–∞, –ø–æ—ç—Ç–æ–º—É –≤—ã –º–æ–∂–µ—Ç–µ –≤—ã–±—Ä–∞—Ç—å ‚Äî –±—É–¥—å —Ç–æ —Ç—Ä–∞—Å—Å–∏—Ä–æ–≤–∫–∞ —Å—Ç–µ–∫–∞, –∫–æ–¥ –æ—à–∏–±–∫–∏ –¥–ª—è –∫–æ–Ω–∫—Ä–µ—Ç–Ω–æ–≥–æ –ø—Ä–∏–ª–æ–∂–µ–Ω–∏—è –∏–ª–∏ –ø—Ä–æ—Å—Ç–æ —Ç–µ–∫—Å—Ç.
